#wenke

> 腾讯问问 & 百科前端自动化构建工具，wenke为"wenwen and baike"的缩写，有“温可”含义，寓为"温柔可人"，对开发同学和项目友好，配置灵活。


##安装
```
npm install -g wenke
```
